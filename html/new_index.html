Domain

BLAST is multi domain compliante, BLAST secures the access of your objects thru a given domain. BLAST supports multi-users domain where each user can safely access the domain content. The domain is the virtual offering to the BLAST users a the security level avoiding any fobidden users to access your domain objects.

Infrastructure

BLAST is multi infrastructure oriented, BLAST simplifies the management of your infrastructure inventories. You will be able to discover quickly new inventory members and shortly add them into an existing infrastructure. With BLAST your infrastructure inventory becomes flexible and easy to manage.

Scheduler

BLAST provides autonomous orchestrator, the orchestator allows you to schedule any management scenario to be run at any time. The scenario will be executed on the scheduler demand then all execution results will be stored for later consultation.

Script

BLAST supports multiple script languages, the common script management interface allows you to run different script languages and interpret them in a common way. The script execution and result can be consulted and displayed via the management interface.
