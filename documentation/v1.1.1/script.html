SCRIPT

The cluster is a Blast object member of the infrastructure sublayer, the cluster can have one or more downlinks to node objects also the cluster can have one more uplinks to infrastructure objects.

Script form

A new cluster can be created by filling the cluster form. Every newly created cluster will be part of the current realm.

  • script file is a mandatory field, it names the script object.
  • script arguments is an optional field, it describes the script object shortly.
  • script description is an optional field, it describes the script object shortly.
  • script lang is an optional field, it describes the script object shortly.
  • script realm is an optional field, it describes the script object shortly.
  • script shareable is an optional field, it describes the script object shortly.

Script list

All clusters of the current realm are visibile as card in the list section. Click on the cluster card to access to the cluster details section.

script details

Click on the cluster card of your choice to open the cluster details. The cluster details contains three sections:

  • cluster details shows the id, name, description, realms.
  • cluster nodes management allows you to manage the cluster nodes.
  • cluster nodes list lists the nodes linked.
