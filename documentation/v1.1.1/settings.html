SETTINGS

The cluster is a Blast object member of the infrastructure sublayer, the cluster can have one or more downlinks to node objects also the cluster can have one more uplinks to infrastructure objects.

Settings menu

A new cluster can be created by filling the cluster form. Every newly created cluster will be part of the current realm.

Settings form - Ansible

A new cluster can be created by filling the cluster form. Every newly created cluster will be part of the current realm.

  • ansible username is a mandatory field, it names your cluster.
  • ansible password is an optional field, it describes your cluster shortly.
  • ansible password set is an optional field, it describes your cluster shortly.
  • ansible certificate is an optional field, it describes your cluster shortly.
  • ansible certificate set is an optional field, it describes your cluster shortly.
  • ansible inventory location is an optional field, it describes your cluster shortly.

Settings form - SSH

All clusters of the current realm are visibile as card in the list section. Click on the cluster card to access to the cluster details section.

  • ssh username is a mandatory field, it names your cluster.
  • ssh password is an optional field, it describes your cluster shortly.
  • ssh password set is an optional field, it describes your cluster shortly.
  • ssh certificate is an optional field, it describes your cluster shortly.
  • ssh certificate set is an optional field, it describes your cluster shortly.
  • ssh inventory location is an optional field, it describes your cluster shortly.
