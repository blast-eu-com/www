NODE

The node is a Blast object member of the cluster sublayer, the node can have one more uplinks to cluster objects.

Node form

A new node can be created by filling the node form. Every newly created node will be part of the current realm.

  • node hostname is a mandatory field, it is the hostname of the node or any alias.
  • node ip is a mandatory field, it is the ip of the node.
  • node description is an optional fields, it describes the nodes.
  • scan by node ip it describes what parameter BLAST will use to access the node. If selected then BLAST will use the node ip else BLAST will use the node hostname.

Node list

All nodes are visibile as card in the list section. Click on the node card to access to the node details section.

Node details

Click on the node card of your choice to access his details. The node details contains three sections:

  • node details shows the id, hostname, ip, description, mode, realms.
  • node mode allows you to manage the cluster nodes.
  • node peers lists the nodes linked.
