SCHEDULER

The scheduler is a Blast object.

Scheduler menu

Scheduler form

A new scheduler can be created by filling the scheduler form.

  • scheduler name is a mandatory field, it names the scheduler object.
  • scheduler description is an optional field, it describes the scheduler object shortly.
  • scheduler days is a mandatory , it defines which days of the week the scheduler must run.
  • scheduler execution time is an optional value, it defines at what time the scheduler must run.
  • scheduler execution interval (min) is an optional value, it defines the interval in minutes between every scheduler execution.
  • scheduler execution interval (sec) is an optinal value, it defines the interval in secondes between every scheduler execution.
  • scheduler parallel mode is an optional selector, the default value is false. when a scheduler has more than one scenario then it defines if the scenarios must be execute in parallel.
  • scheduler scenario is mandatory, it defines one or more scenarios to be run by the scheduler.
