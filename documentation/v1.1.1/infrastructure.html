INFRASTRUCTURE

The infrastructure is a Blast object at the root of the infrastructure tree, the infrastructure can have one or more downlinks to cluster objects.

Infrastructure form

A new infrastructure can be created using the infrastructure form. Every newly created infrastructure will be part of the current realm.

  • infrastructure name is a mandatory field, it names the infrastructure object.
  • infrastructure description is an optional field, it describes the infrastructure object.

Infrastructure list

All infrastructures of the current realm are visibile as card in the list section. Click on the infrastructure card to access to the infrastructure details.

Infrastructure details

Click on the infrastructure card of your choice to go to the details section. The infrastructure details contains three sections:

  • infrastructure details shows the id, name, description, realms.
  • infrastructure clusters management allows you to manage the infrastructure clusters.
  • infrastructure clusters list lists the clusters linked.
